# Ontario Payroll Calculator

Simple Ontario payroll calculator built with React + Vite.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this folder to GitHub
2. Sign in to Vercel with GitHub
3. Import the repo
4. Click Deploy

## Notes

This is an educational estimator, not a production payroll engine.
For real payroll, validate with CRA PDOC and your payroll provider.
